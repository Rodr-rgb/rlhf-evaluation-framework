# RLHF Evaluation Framework

Production-style system for evaluating LLM outputs using modular scoring and API architecture.

## Architecture
- Core scoring engine
- Service layer abstraction
- API routing layer
- Unit tests
- Docker support

## Features
- Prompt-response evaluation
- Pass/fail classification
- Normalized scoring
- Test coverage

## Tech Stack
- Python
- FastAPI
- Pytest
- Docker
